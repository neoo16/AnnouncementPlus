# AnnouncementPlus

Professional Minecraft server plugins designed to enhance server experience with announcements.

---

## Plugins Overview

### AnnouncementPlus
- Broadcast server-wide announcements with titles, subtitles, and sounds.
- Automatically adapts message to fit the screen.
- Supports `/announcement <message>` command.
- Includes a console startup banner with plugin info.

---

## Features
- Easy-to-use commands for admins
- Fully compatible with Paper 1.21.10 servers.
- Minimal performance impact.

## Installation
1. Download the plugin `.jar` file.
2. Place it in your server's `plugins` folder.
3. Start or restart your server.
4. Plugin will automatically enable and display startup info in the console.
